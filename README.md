# 🧠 Medical AI & LLM Evaluation Skill Collection
A professional, reusable skill set for AI product managers, built for 0-to-1 construction of medical AI evaluation systems (ByteDance Xpert benchmark).

---

## 📋 Overview
This repository hosts a curated collection of skills focused on:
- **Xpert Medical Health Benchmark**: 0-to-1 build of medical LLM evaluation system, covering 20+ clinical scenarios
- **LLM General & Expert Evaluation**: HLE/XpertBench (Benz) benchmark design and process standardization
- **End-to-end Evaluation Workflow**: Expert question creation → multi-level quality inspection → cross-acceptance

All skill files are stored in the `Skill/` directory (**no README inside the folder**, strictly following specifications).

---

## 📁 Skill Directory
| Skill File Name | Core Function | Applicable Scenario |
|-----------------|---------------|---------------------|
| `xpert-medical-evaluation.skill` | Xpert medical benchmark 0-to-1 construction standard | Medical AI large model evaluation system building |
| `xpert-question-process.skill` | Question creation, quality inspection, acceptance workflow | Expert-level LLM evaluation process standardization |
| `hle-general-evaluation.skill` | HLE general capability assessment framework | General LLM comprehensive capability evaluation |
